## Project Overview

AutoExpert (aka Semni) is a full-stack web application for taking driving license quizzes. It features:

- Modern React frontend with Firebase Analytics
- Express.js REST API with JWT authentication
- MySQL database on Cloud SQL
- Automated deployments via GitHub Actions
- Infrastructure as Code with Terraform

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Firebase
- **Backend**: Node.js, Express.js, MySQL, Knex.js
- **Infrastructure**: GCP, Terraform, Cloud Run, Firebase Hosting
- **CI/CD**: GitHub Actions, Workload Identity Federation
